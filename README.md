# Bike Store Sales Dashboard & Search Tool

**Goal:** Designed a dynamic inventory search tool and sales analytics dashboard for a retail bike store.

**Files:**
* [Download the Excel File](Bike_Store_Analytics.xlsx)
* ![Search Tool Preview](search_tool_preview.png)
* ![Dashboard Preview](dashboard_preview.png)

**Key Features:**
* **Search Application:** Built a user-friendly lookup tool using `VLOOKUP` and `INDEX-MATCH` to instantly retrieve product pricing, brand, and stock details.
* **Error Handling:** Implemented `IFERROR` logic to ensure a clean user interface when search terms are missing.
* **Interactive Dashboard:** Created Pivot Tables and Slicers to analyze "Products per Brand" and "Average Price" trends.
* **Data Cleaning:** Sanitized raw dataset by removing non-breaking spaces and correcting currency formatting.
